# 🎲 QuấtWin - Game Tài Xỉu

Game cờ bạc trực tuyến với giao diện đẹp mắt và hiệu ứng sống động.

## 🎮 Cách chơi

1. **Chọn cửa:** Tài (11-18) hoặc Xỉu (3-10)
2. **Nhập tiền cược:** 
   - Số: 1000, 5000
   - K/M/B: 1k, 2m, 3b  
   - Phần trăm: 20%, 50%
3. **Tung xúc sắc** và chờ kết quả!

## 🎁 Phần thưởng

- **Thắng thường:** x1
- **Trùng số:** x3 (3 xúc sắc cùng số)
